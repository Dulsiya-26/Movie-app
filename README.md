# Movie-app
🎬 Movie App (React + Router)  A simple and responsive Movie App built using React.js and React Router. This project demonstrates basic React concepts such as components, props, state management, and routing between pages.

🚀 Features
🎥 Display list of movies
🔍 Movie details page
🔄 Navigation using React Router
🧩 Reusable React components
📱 Basic responsive UI

🛠️ Tech Stack
React.js
React Router DOM
JavaScript (ES6+)
HTML & CSS
📌 Purpose

This project is created for learning and practicing React fundamentals and routing concepts.
